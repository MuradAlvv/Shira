## Technologies used
* Java 17
* Spring Boot, Security, Data, Cloud
* Apache Kafka
* WebSocket
* PostgreSQL
* Gmail API(OAuth2)
* Docker

## Run

Run the following command from the root folder

```bash
docker-compose up --build 
```
